# Диаграммы проекта StoreAdmin

В этой папке должны находиться диаграммы архитектуры проекта:

## UML диаграммы
- Диаграммы классов
- Диаграммы последовательностей
- Диаграммы компонентов
- Диаграммы развертывания

## DFD (Data Flow Diagram)
- Диаграммы потоков данных
- Иерархия диаграмм DFD

## ER диаграммы
- ER диаграммы для PostgreSQL
- ER диаграммы для MySQL
- ER диаграммы для MongoDB

Для создания диаграмм можно использовать инструменты:
- draw.io
- Lucidchart
- PlantUML
- Mermaid
